 # Build a Responsive Navbar

## ✅ Open your VS Code and complete this assignment manually.

## ❌ Do NOT use any AI editor, AI autocomplete, or AI-generated code inside VS Code.
You must write the code yourself.

## 🧩 Problem Statement
Create a fully responsive navigation bar for a web app using React.

The navbar should look clean on desktop screens and collapse into a hamburger menu on smaller screens (tablet and mobile).

You must follow the design exactly:

#### Desktop: Horizontal navbar with Logo, four menu items, and a "Create Account" button.

#### Tablet: Full-width green background menu with an "X" close icon. Menu items should remain horizontal.

#### Mobile: Full-screen green vertical menu with centered items and "Create Account" at bottom.
